🎬 YouTube Clone

A modern YouTube Clone built using React.js, Redux Toolkit, and Tailwind CSS. This project replicates core features of YouTube, allowing users to search, browse, and watch videos in real time using API integration.

🚀 Features
🔍 Search videos
🎥 Watch videos
📺 Channel details
⚡ Real-time API data
📱 Responsive UI
🛠️ Tech Stack
React.js
Redux Toolkit
Tailwind CSS
RapidAPI
⚙️ Setup
git clone https://github.com/ShakirFarhan/Youtube-Clone.git
cd Youtube-Clone
npm install

Create .env:

VITE_RAPID_API_KEY=your_api_key
VITE_RAPID_API_HOST=youtube-v31.p.rapidapi.com

Run:

npm start
🌐 Usage

Open http://localhost:3000 → Search → Watch videos

🎯 Objective

To learn frontend development, API integration, and state management by building a real-world app.
